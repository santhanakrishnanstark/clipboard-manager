Usage Guide

Learn how to use all the features of Clipboard Manager effectively

Getting Started

Once installed, Clipboard Manager works automatically in the background. Here's how to get started:

  1. Copy any text from any website (Ctrl+C or Cmd+C)
  2. Click the Clipboard Manager icon in your Chrome toolbar
  3. See your copied text appear in the clipboard history
  4. Click any item to copy it back to your clipboard
Tip: The extension automatically captures all copied text across all websites, so you don't need to do anything special to start building your clipboard history.

Basic Operations

Copying Items Back to Clipboard

  • Click any item in the history to copy it to your clipboard
  • The item will be highlighted briefly to confirm the action
  • You can then paste it anywhere using Ctrl+V (or Cmd+V)

Searching Your History

  • Use the search bar at the top of the popup
  • Type keywords to filter your clipboard history
  • Search is case-insensitive and matches partial text
  • Clear the search to see all items again

Deleting Items

  • Hover over any item to see the delete button (×)
  • Click the delete button to remove the item permanently
  • Use "Clear All" to remove all clipboard history

Snippets Management

Save frequently used text as snippets for quick access:

Creating Snippets

  1. Click the "Snippets" tab in the popup
  2. Click "Add New Snippet"
  3. Enter a name and the text content
  4. Click "Save" to create the snippet

Using Snippets

  • Click any snippet to copy it to your clipboard
  • Use the search bar to find specific snippets
  • Edit snippets by clicking the edit icon
  • Delete snippets using the delete button

Settings & Configuration

Customize Clipboard Manager to fit your workflow:

Accessing Settings

  • Click the gear icon (⚙️) in the popup
  • Or right-click the extension icon and select "Options"
  • Or go to chrome://extensions and click "Details" → "Extension options"

General Settings

  • Maximum History Size: Set how many items to keep (10-1000)
  • Enable Context Menu: Add clipboard options to right-click menus
  • Enable Quick Paste: Allow floating quick paste menu
  • Theme: Choose Auto (system), Light, or Dark mode
  • Auto-clear: Automatically clear old items after specified days

    Import & Export

    Backup and restore your clipboard data:

    Exporting Data

    1. Go to Settings → Data Management
    2. Click "Export Data"
    3. Choose format (JSON or CSV)
    4. Save the file to your computer

    Importing Data

    1. Go to Settings → Data Management
    2. Click "Import Data"
    3. Select your backup file
    4. Choose merge or replace options
    Backup Tip: Export your data regularly to avoid losing important clipboard history and snippets.

    Context Menu Integration

    Access clipboard features from right-click menus:

    Available Options

    • Copy to Clipboard Manager: Manually add selected text
    • Paste from History: Quick access to recent items
    • Paste Snippet: Insert saved snippets
    • Open Clipboard Manager: Launch the full interface

    Enabling Context Menu

    Enable this feature in Settings → General → Enable Context Menu

    Quick Paste Menu

    Access clipboard items with a floating menu:

    Using Quick Paste

    1. Press Ctrl + Shift + V (or ⌘ + Shift + V on Mac)
    2. A floating menu appears with recent clipboard items
    3. Click an item or use arrow keys to navigate
    4. Press Enter to paste the selected item
    Pro Tip: Quick Paste is perfect for rapid text insertion without opening the full popup.

    Best Practices

    Organizing Your Workflow

    • Use descriptive names for snippets
    • Organize snippets by category or purpose
    • Regularly clean up old clipboard items
    • Set appropriate history size for your needs
    • Use excluded sites for sensitive pages

    Privacy & Security

    • Add banking and financial sites to excluded sites
    • Regularly clear sensitive data from history
    • Use the auto-clear feature for automatic cleanup
    • Be mindful of what you copy on shared computers

    Performance Tips

    • Keep history size reasonable (50-200 items)
    • Clear old items periodically
    • Avoid copying very large text blocks repeatedly
    • Use snippets for frequently accessed content

    Advanced Features

    Pinning Items

    Keep important items at the top of your history:

    • Click the star icon next to any clipboard item
    • Pinned items appear at the top of the list
    • Pinned items are protected from auto-cleanup
    • Click the star again to unpin

    Sorting Options

    Organize your clipboard history:

    • Newest First: Most recent items at the top (default)
    • Oldest First: Oldest items at the top
    • Pinned First: Pinned items, then by date
    • Alphabetical: Sort items alphabetically

    Bulk Operations

    • Select multiple items using checkboxes
    • Delete multiple items at once
    • Export selected items only
    • Pin/unpin multiple items
