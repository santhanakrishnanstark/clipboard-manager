Installation Guide

Follow these simple steps to install the Clipboard Manager Chrome extension

Method 1: Chrome Web Store (Recommended)

Coming Soon: The extension will be available on the Chrome Web Store once it's published.
  1. Visit the Chrome Web Store page (link coming soon)
  2. Click the "Add to Chrome" button
  3. Confirm the installation by clicking "Add extension"
  4. The extension icon will appear in your Chrome toolbar

Method 2: Manual Installation (Developer Mode)

For early access or development purposes, you can install the extension manually:

Step 1: Download the Extension

Download the latest release from GitHub:

Download ZIP File

Or clone the repository:

git clone https://github.com/yourusername/clipboard-manager.git

Step 2: Extract Files

  1. Extract the downloaded ZIP file to a folder on your computer
  2. Remember the location of this folder - you'll need it in the next step

Step 3: Enable Developer Mode

  1. Open Google Chrome
  2. Navigate to chrome://extensions/
  3. Toggle on "Developer mode" in the top-right corner

Screenshot: Chrome Extensions page with Developer mode toggle

Step 4: Load the Extension

  1. Click "Load unpacked" button
  2. Navigate to and select the extracted extension folder
  3. Click "Select Folder" (or "Open" on Mac)
  4. The extension should now appear in your extensions list

Step 5: Pin the Extension

  1. Click the puzzle piece icon in Chrome's toolbar
  2. Find "Clipboard Manager" in the list
  3. Click the pin icon to pin it to your toolbar

Verify Installation

To confirm the extension is working correctly:

  1. Look for the Clipboard Manager icon in your Chrome toolbar
  2. Copy some text from any webpage
  3. Click the extension icon to open the popup
  4. You should see your copied text in the clipboard history
Success! If you can see your copied text, the extension is working correctly.

System Requirements

  • Browser: Google Chrome 88+ or Chromium-based browsers
  • Manifest: Supports Manifest V3
  • Permissions: Storage, Active Tab, Context Menus, Clipboard
  • Storage: Minimal local storage usage (typically < 1MB)

Required Permissions

The extension requires the following permissions to function:

Storage

To save clipboard history and settings locally on your device

Active Tab

To capture copied text from the current webpage

Context Menus

To add clipboard options to right-click menus

Clipboard Read/Write

To read copied text and write text back to clipboard

Installation Troubleshooting

Extension Not Loading

  • Ensure you've enabled Developer mode
  • Check that you selected the correct folder (should contain manifest.json)
  • Try refreshing the extensions page and loading again
  • Check Chrome's console for any error messages

Extension Icon Not Visible

  • Click the puzzle piece icon in Chrome's toolbar
  • Look for "Clipboard Manager" and click the pin icon
  • The icon should now appear in your toolbar

Not Capturing Clipboard Data

  • Refresh the webpage and try copying again
  • Check if the site is in your excluded sites list
  • Ensure clipboard permissions are granted
  • Try disabling and re-enabling the extension
