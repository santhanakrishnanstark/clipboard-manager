Privacy & Security

Control which sites can access your clipboard and manage privacy settings.

Excluded Sites
Sites where clipboard data should not be captured (one per line)
• Use exact domain names (e.g., example.com)
• Use wildcards for subdomains (e.g., *.example.com)
• Banking and medical sites are recommended for exclusion

Data Management

Import, export, and manage your clipboard data and settings.

Export Data
Download your clipboard history and snippets
Import Data
Import clipboard history and snippets from file
Clear All Data
Remove all clipboard history and snippets

Keyboard Shortcuts

Keyboard shortcuts for quick access to clipboard features.

Open Clipboard Manager
⌘+Shift+C (Mac) / Ctrl+Shift+C (Windows/Linux)
Quick Paste Menu
⌘+Shift+V (Mac) / Ctrl+Shift+V (Windows/Linux)
Paste Last Item
⌘+Shift+Z (Mac) / Ctrl+Shift+Z (Windows/Linux)
